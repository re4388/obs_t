通用語法
```sql

SELECT [DISTINCT] <column expression list> 
FROM <single table> 

[WHERE <predicate>]

[GROUP BY <column list>
 [HAVING <predicate>]]

[ORDER BY <column list>] 
[LIMIT <integer>];


ex

select user, mail, level
from user_table
where user_id like "ben%"
group by level
having level > 3
order by id
limit 1


```
