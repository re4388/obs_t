# m3u8 知識
- [(385) 通过 M3U 免费看电视直播？先来搞懂什么是 M3U8 - YouTube](https://www.youtube.com/watch?v=KS0-57ZDyy4&list=WL&index=4)
- m3u 純文本
- m3u8: utf8 encode 的 m3u
- 都是 HLS (http live stream)協議的一部分
- 把整個檔案拆成一個個小的 http 來下載
- server encode 為 movie 檔案
- 切小的，透過 HLS 傳
- 切成一個一個ts檔案，一個一個傳
    
    看一下格式
    格式
    幾秒開始
    還有每一個地址
    
    test.m3u
    ```YAML
    \#EXTM3U
    \#EXTINF:0,test
    https://node1.olelive.com:5443/live/CCTV1HD/hls.m3u8
    \#EXTINF:1000,test2
    https://node1.olelive.com:5443/live/CCTV2HD/hls.m3u8
    ```
    
    下面這個可以直接導到 vlc播放測試
