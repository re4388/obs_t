
- string (UTF-8 編碼)
- 自用**不會用 $ 開頭**（“$SYS” 是 MQTT 伺服器的控制介面主題的保留字）
- **不可包含 #和 + 字元**
- 減號(-)和乘號（*）在程式語言中有特殊意義，為了避免誤會，也不建議使用。
- 名稱的**英文大小寫有區別**，home 和 Home 是兩個不同的名稱。
- 支援類似檔案路徑的階層式命名方式
